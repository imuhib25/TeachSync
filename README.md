# TeachSync – Tuition Tracker 📘

TeachSync is a simple and efficient **Tuition Management & Tracking App** built to help tutors, private teachers, and coaching centers manage students, attendance, and fees with ease.

The app is designed to reduce paperwork and keep all tuition-related records organized in one place.

---

## 🚀 Features

- 👨‍🎓 **Student Management**  
  Add, edit, and manage student profiles effortlessly.

- 🏫 **Batch / Class Management**  
  Organize students into batches or classes.

- 🗓️ **Attendance Tracking**  
  Mark daily attendance and maintain attendance history.

- 💰 **Fee & Payment Tracking**  
  Track monthly fees, paid amounts, and pending dues.

- ⏰ **Due History & Records**  
  View past payments and outstanding balances clearly.

- 📊 **Simple & Clean UI**  
  Easy-to-use interface for fast daily operations.

---

## 🛠️ Tech Stack

- **Platform:** Android  
- **Language:** Java
- **UI:** XML  
- **Database:** Firebase
